Comment jouer ?

Distribution

Chaque joueur reçoit un mot secret. Les civils ont le même mot, l'undercover a un mot proche.

💬

Description

Tour à tour, chaque joueur donne un indice sur son mot. Soyez subtil !

🗳️

Vote

Tous votent pour éliminer le suspect. Le plus voté est éliminé.

🏆

Victoire

Les civils gagnent en éliminant tous les undercovers. L'undercover gagne s'il reste jusqu'à la fin.
